## 📝 Deskripsi Proyek
Nama aplikasi (Cilegon Juare) playstore
Repositori ini berisi dokumentasi dan implementasi praktis untuk Ujian Tengah Semester mata kuliah Big Data. Proyek ini mencakup pengolahan data menggunakan ekosistem Big Data untuk memproses, menganalisis, dan memvisualisasikan dataset dalam skala besar.

## 🚀 Fitur Utama
* **Data Ingestion:** Proses pengambilan data dari sumber eksternal.
* **Data Processing:** Pembersihan dan transformasi data menggunakan framework pengolahan terdistribusi.
* **Exploratory Data Analysis (EDA):** Analisis statistik untuk menemukan pola dalam data.
* **Visualization:** Representasi grafis dari hasil analisis untuk mempermudah pengambilan keputusan.
